Projet Météo

Le projet météo est la création d'une application gérant des données météorologiques et les traitants.

Les objectifs
Le projet météo est composé en 3 grandes parties :
1. Récuperer les données météorologiques. Pour faire ceci, l'autre membres de mon équipe gérer de nombre capteur est une transmission bluetooth pour relier les données qu'il récupère à l'ordinateur possédant le logiciel.
2. Stocké les données récuperées dans une base de données.
3. Utilisé les données pour obtenir des statistiques. Par exemple : affiché des moyennes de température sur un certain temps.
Evedement, toutes les informations (statistiques et données actuelles) sont affichés sur une IHM faite sous Qt
