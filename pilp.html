PILP

PILP est un projet personnel ayant pour but de gérer les publications d'un compte Instagram.

La Problèmatique
J'utilise un compte instagram pour données mon avis sur des mangas et des comics.
Après chaque lecture, je note mon avis et je note sur un Excel que je l'ai lu (il me sert principalement pour gérer les postes mais j'en sors aussi quelques statistiques).


Les objectifs
Le projet PILP est là pour regrouper les avis, le fichier Excel et la création d'une PAL (Pile A Lire) et pouvoir post directement.
Le nom PILP n'est pas confirmé, PILP s'ignifie Post Instagram Liste Pal


Les fonctionnalité
- Création d'un poste associé à une série / tome présent dans la PAL (création d'un poste avec un preset créer automatiquement et enlever de la PAL)
- Création d'un poste (création d'un poste avec un preset "par défaut")
- Création d'un poste automatique pour chaque fin de mois pour un bilan lecture (création d'un poste avec un preset "bilan lecture") avec les tomes que j'ai lus durant ce mois
- Ajoute d'un tome dans la PAL
- Publication d'un poste
- d'autres fonctionnalité reste à être définie.

Les technologies
Pour ce qui est des technologies utilisées, je pense partir sur des technologies du web (Node JS et React JS) pour pouvoir gérer mon compte de n'importe qu'elle ordinateur ou mobile (avec la création possible d'une PWA).
Pour ce qui est de la base de données, cela dépendra de la technologie utilisée mais sûrement du PostgreSQL avec Heroku.
Ces technologies ne sont pas confirmées car la projets a été mis en pause pour ce concentré sur des projets plus importants.
