Description

Parcourez un monde vaste pour découvrir le mystère qui rôde.
Platformer 2D sur PC fais avec Unity seul. Ce projet a pour but de m'apprendre la programmation d'un jeu vidéo et Unity.
J'ai utilisé les tutoriels de TutoUnityFR et la documentation d'Unity pour ce projet.
